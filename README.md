## 🚀 Featured Projects

### 🍸 [POV Cocktail Video Generator](https://github.com/KHOUB-ENG/pov-cocktail-video-generator)
A **multi-agent LLM orchestration pipeline** that turns a one-line brief into a complete
short-form-video production package, via **8 sequential Google Gemini agents** behind a Flask
dashboard — with retry/backoff, live progress streaming, per-agent generation tuning, and
resumable session state.
`Python` · `Flask` · `Google Gemini` · `SQLite` · `Multi-agent orchestration` · `pytest` · `GitHub Actions`
